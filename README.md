# UNFLOOD

UNFLOOD is a cross-platform IoT-based flood early warning system developed using .NET MAUI. This application integrates ESP32 telemetry, SQLite local database, Hybrid Neural-Fuzzy Decision System, BMKG API, OpenCV Face Recognition, and Gemini AI for real-time flood monitoring and emergency mitigation.

## Features

- Real-time flood monitoring
- ESP32 telemetry integration
- Hybrid Neural-Fuzzy flood classification
- SQLite local database logging
- BMKG weather API integration
- OpenCV face recognition login
- Gemini AI mitigation assistant
- Role-based dashboard for Warga, BMKG, and Admin

## Technologies

- .NET MAUI
- C#
- SQLite
- ESP32
- OpenCV
- Gemini API
- BMKG API
- Neural-Fuzzy Logic
